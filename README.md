# Ireland's Data Centre Investigation Web
### The Ripple Rebellion — Bríd | Midleton, Co. Cork, Ireland

A living investigative map of Ireland's data centre regulatory 
architecture, built and maintained by The Ripple Rebellion.

## What this is

An interactive web documenting the five regulatory instruments 
passed between 2025-2026 that together form a layered architecture 
making meaningful environmental accountability almost impossible 
to trigger — with no public debate worthy of the name.

## What it covers

- The Five Instruments (Acts 1-5)
- Regulatory loopholes including the Grandfather Clause 
  and Fuel Type Gap
- The CRU ruling CRU/2025/236
- The LEAP Large Energy User Action Plan January 2026
- Global context — Ireland projected as world's 3rd largest 
  data centre hub
- Cork Harbour 50-year projection
- NDAs, thermal discharge and brine silence
- Organoid Intelligence regulatory void
- Actions — AIE requests, outreach, AI testimony

## This is a living document

It will be updated as the investigation develops. 
New videos, AI testimony, Oireachtas debates, 
and international comparisons will be added over time.
